# Banking Loan Data Analysis using SQL

## Project Overview
This project focuses on cleaning, transforming, and analyzing banking loan data using MySQL.

## Features
- Data Cleaning
- Date Format Conversion
- Loan Trend Analysis
- State-wise Reporting
- Payment Analysis
- Grade-wise Balance Analysis

## SQL Concepts Used
- Aggregate Functions
- GROUP BY
- CASE Statements
- Date Functions
- Views
- Stored Procedures
- Data Transformation

## Technologies Used
- MySQL
- SQL
